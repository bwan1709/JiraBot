Lấy thông tin tại Atlassian API Tokens
💡 Hướng dẫn cấu hình kết nối Jira
1️⃣ Lấy Jira API Token
  1. Truy cập trang quản lý mã của Atlassian: Atlassian API Tokens.
  2. Nhấn nút Create API token.
  3. Nhập nhãn tùy ý (ví dụ: JiraBot) rồi nhấn Create.
  4. Sao chép mã API Token hiển thị (lưu ý: mã này chỉ xuất hiện 1 lần duy nhất) và dán vào ô Jira API Token.
2️⃣ Lấy Account ID
  1. Truy cập trang cá nhân của bạn bằng cách click vào ảnh đại diện ở góc trên bên phải, chọn Profile (hoặc chọn People trên thanh menu bên trái và nhấp vào chính mình).
  2. Xem đường dẫn (URL) trên thanh địa chỉ trình duyệt. URL lúc này sẽ hiển thị định dạng:
    https://home.atlassian.com/o/.../people/712020:a1b2c3d4-e5f6-7a8b-9c0d-e1f2a3b4c5d6?cloudId=...
  3. Sao chép chuỗi mã nằm sau chữ /people/ và đứng trước dấu hỏi chấm ? rồi dán vào ô Account ID.
3️⃣ Lấy Cloud ID
  1. Xem trực tiếp tại đường dẫn (URL) trang Profile cá nhân của bạn ở Bước 2. Nó có chứa thuộc tính cloudId ở ngay cuối:
    https://home.atlassian.com/...people/...?cloudId=11223344-5566-7788-9900-aabbccddeeff
  2. Sao chép chuỗi mã UUID ở phía sau dấu cloudId= ở cuối URL và dán vào ô Cloud ID.
  3. Cách dự phòng: Mở tab mới, truy cập đường dẫn sau khi đã đăng nhập Jira: https://<tên_công_ty>.atlassian.net/_edge/gira/profiles rồi sao chép mã "id" từ kết quả hiển thị.
4️⃣ Jira Base URL
  • Đường dẫn chính dùng để truy cập vào Jira của bạn.
  • Định dạng chuẩn: https://<tên_công_ty>.atlassian.net (lưu ý: không thêm dấu gạch chéo / ở cuối).
